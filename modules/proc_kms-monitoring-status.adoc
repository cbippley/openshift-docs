= Monitoring KMS encryption status

[role="_abstract"]
Monitor KMS encryption status using operator and API server logs to verify encryption is functioning correctly and detect issues early.

.Procedure

. Check the kube-apiserver operator logs for KMS-related events:
+
[source,terminal]
----
$ oc logs -n openshift-kube-apiserver-operator deploy/kube-apiserver-operator | grep -i kms
----

. View API server logs for KMS-related events:
+
[source,terminal]
----
$ oc logs -n openshift-kube-apiserver -l apiserver=true --tail=100 | grep -i kms
----

. Verify the KMS encryption configuration:
+
[source,terminal]
----
$ oc get apiserver cluster -o jsonpath='{.spec.encryption}' | jq
----
+
Output should show encryption type as `KMS`.
