= Available KMS providers

[role="_abstract"]
{product-title} supports any KMS v2-compatible plugin to allow integration with your existing key management infrastructure.

== Recommended provider

*HashiCorp Vault*

Use HashiCorp Vault Transit secrets engine for KMS encryption. Requires Vault instance with Transit engine enabled and network access from control plane nodes.

== Custom providers

Custom KMS plugins must implement the Kubernetes KMS v2 protocol. You deploy and manage custom plugins on control plane nodes.
